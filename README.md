# Aplikasi Arisan

Aplikasi berbasis Java Desktop (Swing) yang dirancang untuk mengelola dan memutar undian arisan secara digital. Aplikasi ini dikembangkan menggunakan NetBeans IDE dan mendukung visualisasi undian yang interaktif.

## âœ¨ Fitur Utama
* **Manajemen Peserta:** Tambah, hapus, dan lihat daftar anggota arisan.
* **Sistem Undian:** Fitur putar undian (Roda Keberuntungan) untuk menentukan pemenang secara acak.
* **Riwayat Pemenang:** Mencatat siapa saja yang sudah memenangkan undian di setiap periode.
* **Antarmuka Interaktif:** Menggunakan komponen kustom seperti `PanelRoda` untuk pengalaman pengguna yang lebih baik.
